A place to get started with coding!
Our program teaches basic computer programming to help students get a taste of what coding is like. We're hoping to provide students the opportunity to get started in learning about programming even if they decide to follow another path.
Students will learn basic programming such as using the terminal and setting up github.
Students will learn the basic syntax of Python and how to write their first programs.
Students will learn how to write functions and combine them to create a dynamic program.
